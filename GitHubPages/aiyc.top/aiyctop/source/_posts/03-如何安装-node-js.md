*   词汇结构
*   表达式
*   数据类型
*   变量
*   函数
*   this
*   箭头函数
*   循环
*   作用域
*   数组
*   模板字面量
*   分号
*   严格模式
*   ECMAScript 6、2016、2017

具备这些概念，无论是在浏览器还是在 Node.js 中，都会成为一名熟练的 JavaScript 开发者。 以下概念也是理解异步编程的关键，异步编程是 Node.js 的基本组成部分：

*   异步编程与回调
*   定时器
*   Promise
*   Async 与 Await
*   闭包
*   事件循环
